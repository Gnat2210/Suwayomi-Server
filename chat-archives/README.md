# Chat Archives

This directory stores local copies of Copilot chat sessions and related artifacts so they can be accessed without the chat UI.

## Structure

- One folder per archived session: `YYYY-MM-DD_<session-id>`
- `chat-session-resources/`: tool-call output capture files from the session
- `debug-logs/`: Copilot debug logs for the session
- `transcripts/`: chat transcript exports (`.jsonl`)
- `notes/`: human-readable summaries and context
- `memory/`: snapshots of memory notes captured during the session
